✈️ 손안에 가이드

AI가 들려주는 여행 이야기 - 파리 현장 가이드가 만든 PWA

🎯 이 앱으로 무엇을 할 수 있나요?

여행지에서 사진만 찍으면 AI가 자동으로 설명을 작성하고,
음성으로 읽어줍니다. 카메라 버튼 하나로 모든 게 끝!

📱 3단계 사용법

1

📸 사진 촬영하기

화면 아래 카메라 버튼을 누르세요. 카메라가 켜지면 여행지를 향해 촬영 버튼을 누릅니다.

2

🤖 AI가 자동 설명

잠깐 기다리면 AI가 사진을 분석해서 텍스트 설명을 만들어줍니다. 화면에서 바로 읽을 수 있어요!

3

🔊 음성으로 듣기

▶ 재생 버튼을 누르면 AI가 설명을 음성으로 읽어줍니다. 눈이 피곤할 때 귀로 들으세요!

🔘 주요 버튼 설명

📷
카메라 사진 촬영
▶
재생 음성 듣기
💾
저장 보관함에 저장
📁
보관함 저장한 사진 보기
📤
공유 친구에게 전송
🗑️
삭제 사진 지우기

⭐ 추천 콘텐츠 (Featured Gallery)

메인 화면 위쪽에 큰 글씨로 된 제목이 보이나요?
클릭하면 파리 가이드가 직접 선별한 명소 설명을 볼 수 있습니다!
다운로드 수가 많은 인기 콘텐츠를 확인하세요.

💡 꿀팁

  • 음성이 안 나올 때: 휴대폰 무음 모드를 해제하고 볼륨을 올려보세요
  • 사진을 여러 장 찍고 싶을 때: 저장 버튼을 누르면 보관함에 저장되고, 다시 카메라를 사용할 수 있어요
  • 오디오는 짧게: 여행 중에는 1분 이상 듣기 어려우니 핵심만 빠르게 들으세요
  • 인터넷 연결 필수: AI 분석을 위해 와이파이나 데이터가 필요합니다
  • 친구에게 공유: 보관함에서 선택 → 공유 버튼으로 카카오톡/문자로 전송 가능
